# Kraken

API GraphQL Node.js Express

## Requirements

* [Nodejs](https://nodejs.org/)

### Install dependencies

```bash
$ npm install
```

### Run API 

```bash
$ npm start
```

### Run API on local environment

```bash
$ npm run local
```

API will be listening on

```bash
$ http://localhost:3000
```

### Run lint

```bash
$ npm run lint
```

### Unity Tests

```bash
$ npm run test
```
